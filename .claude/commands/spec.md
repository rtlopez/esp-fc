# /spec - Create ticket spec.md

Create a ticket-level specification document for implementation.

## Usage

```bash
/spec [$001|feature-slug|TICKET-DIR|SPEC-PATH]
```

## Forwarding

Invoke the `spec` skill and follow `.claude/skills/spec/SKILL.md`.

## Inputs

- Ticket identifier, ticket directory, or `spec.md` path

## Output

- `artifacts/tickets/{ID}/spec.md`
