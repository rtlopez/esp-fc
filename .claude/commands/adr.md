# /adr - Architecture Decision Record

Create an ADR for a project-level or cross-cutting technical decision.

## Usage

```bash
/adr [decision title]
```

## Forwarding

Invoke the `adr` skill and follow `.claude/skills/adr/SKILL.md`.

## Inputs

- Decision title
- Optional context and alternatives

## Output

- `docs/adr/ADR-<NNN>-<short-title>.md`
